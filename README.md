# EcoSpace Energy

Sistema inteligente para monitoramento energético de uma missão espacial experimental.

## Funcionalidades

- Monitoramento de energia solar
- Monitoramento das baterias
- Controle de temperatura
- Monitoramento da comunicação
- Geração automática de alertas
- Tomada de decisão automatizada

## Tecnologias

- Python 3

## Como executar

python ecospace.py
